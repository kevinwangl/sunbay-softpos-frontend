## 🎨 设计特点

### Apple风格设计
- **简洁大气**：去除多余装饰，突出内容
- **圆角卡片**：12px圆角，柔和视觉
- **微妙阴影**：轻量级阴影效果
- **优雅字体**：-apple-system字体栈
- **流畅动画**：自然的过渡效果

### 主题色系
- **主色**：#FF6000（橙色）- 品牌色
- **成功**：#52c41a（绿色）
- **警告**：#faad14（黄色）
- **错误**：#ff4d4f（红色）
- **信息**：#1890ff（蓝色）

### 响应式设计
- **桌面端**（≥1920px）：完整布局
- **平板端**（768-1024px）：自适应布局
- **移动端**（<768px）：折叠菜单，优化触控

## 📦 技术栈

### 前端核心
- React 18.2.0
- TypeScript 5.2.2
- Vite 5.0.8

- Ant Design 5.12.0
- ECharts 5.4.3 + echarts-for-react

### 状态管理
- Zustand 4.4.7（全局状态）

- Axios 1.6.2

## 🚀 快速开始

### 1. 安装依赖
```bash
cd sunbay-softpos-frontend
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

访问：http://localhost:5173

### 3. 登录系统
- 用户名：`admin`
- 密码：`admin123`

### 4. 功能导航
- **仪表板**：/dashboard - 查看整体概况
- **设备管理**：/devices - 管理所有设备
- **威胁事件**：/threats - 查看和处理威胁
- **交易记录**：/transactions - 查询交易历史
